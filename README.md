# neurosync

## Getting Started

### Activating the Virtual Environment

-   enter the virtual environment with `.venv\Scripts\activate` (Windows) or `source .venv/bin/activate` (Mac)
-   exit the virtual environment with `deactivate`

### Installing and Adding Dependencies

-   make sure you are in the virtual environment
-   install existing dependencies using `pip install -r requirements.txt`
-   add any additional dependencies as needed
-   if new dependencies were added, save them to the requirements file with `pip freeze > requirements.txt`
